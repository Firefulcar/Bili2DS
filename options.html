语音识别

⚠️ 这是可选配置。只有当 B 站官方未提供 AI 字幕时，才会自动启用自备的 ASR（语音识别）作为后备方案。绝大多数视频已有官方字幕，无需配置此项。

操作指南

▲ 扩展使用演示

⚠️ 使用前提：本插件需要 B站网页端 和 DeepSeek 网页端（chat.deepseek.com） 均处于登录状态，否则字幕获取或 AI 对话将无法正常工作。

按钮说明

按钮功能
复制字幕获取字幕或语音识别后复制到剪切板
齿轮图标打开本设置页面（右上角绿钩 = 语音识别已就绪）
复述模式完整复述视频内容，转写为流畅文章
快速模式简洁中文列出要点，快速了解视频内容
专家模式五维度深度分析，自动切换 DS 专家模式

固定窗口

  • 开启后多个视频共用同一个 DeepSeek 对话标签页
  • 关闭后每次点击都开启全新对话
  • 每个模式可独立设置

语音识别

  • 视频无字幕时自动启用，需配置 Deepgram API Key
  • 流程：下载音频 → 解码 → Deepgram 中文识别 → 输出 [MM:SS] 文本
  • 费用约 $0.006/分钟，10 分钟视频 ≈ ¥0.4

提示词变量

  • {{text}} — 字幕全文（含时间戳）
  • 修改后点击保存即时生效
  • 点击「恢复默认」回到初始值

使用前提

  • B站网页端需保持登录状态，扩展通过你的登录态获取视频字幕
  • DeepSeek 网页端需保持登录态，需在浏览器中已登录 chat.deepseek.com

常见问题

  • 复制失败？ 切回视频页再试，确保页面有焦点
  • 识别为空？ 视频无语音时结果为空属正常
  • 下载 403？ 自动尝试备用 CDN 地址
  • DS 没反应？ 确认已登录 DeepSeek
